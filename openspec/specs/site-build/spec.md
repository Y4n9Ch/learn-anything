## ADDED Requirements

> **Note:** Updated to reflect actual implementation — site is now pre-built via `vite build` into `site-dist/` and shipped in the npm package. No `files.ts` embedding required.

### Requirement: Build script builds site with Vite and copies to site-dist

The system SHALL provide a build script (`scripts/bundle-site.mjs`) that: (1) runs `vite build` in `packages/cli/site/` to produce `site/dist/`, (2) copies `site/dist/` contents + `site/serve.mjs` to `packages/cli/site-dist/`, and (3) writes `.gitignore` to `site-dist/`.

#### Scenario: Build script produces site-dist

- **WHEN** the build script runs
- **THEN** `site-dist/` contains all built assets (`index.html`, `assets/*.js`, `assets/*.css`), `serve.mjs`, and `.gitignore`

#### Scenario: Build script clears previous output

- **WHEN** the build script runs and `site-dist/` already exists
- **THEN** the previous `site-dist/` is removed before copying new files

#### Scenario: serve.mjs consistency check

- **WHEN** the build script copies `serve.mjs` from `site/` to `site-dist/`
- **THEN** the script verifies both copies have identical content and exits with error if they diverge

### Requirement: site-dist is gitignored and generated at publish time

The system SHALL NOT commit `site-dist/` to version control. It SHALL be generated by the `prepublishOnly` script and included in the npm package via the `files` field in `package.json`.

#### Scenario: site-dist excluded from git

- **WHEN** `git status` is checked in the repo
- **THEN** `site-dist/` is not tracked and listed in `.gitignore`

#### Scenario: site-dist included in npm package

- **WHEN** `npm pack` is run
- **THEN** the tarball includes `site-dist/index.html`, `site-dist/serve.mjs`, and all `site-dist/assets/` files

### Requirement: Build script is integrated into the CLI build pipeline

The system SHALL run the bundle script as the first step of `pnpm build` (before `tsc`) in the CLI package.

#### Scenario: pnpm build runs site bundling

- **WHEN** `pnpm build` or `pnpm -F learn-anything-cli build` is executed
- **THEN** the site bundling script runs before TypeScript compilation

### Requirement: Build script uses Node.js built-in APIs only

The system SHALL implement the build script using Node.js built-in modules (`fs`, `path`, `child_process`). The `vite` CLI is invoked as a subprocess; no additional npm dependencies are required beyond the project's existing devDependencies.
