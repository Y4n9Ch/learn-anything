# Language Basics - Session Notes (2026-06-13)

## Operators and Control Flow

Reviewed JavaScript operators and control structures:

### Operators

- **Arithmetic**: `+`, `-`, `*`, `/`, `%`, `**` (exponentiation)
- **Comparison**: `==` (loose), `===` (strict), `!=`, `!==`, `>`, `<`, `>=`, `<=`
- **Logical**: `&&` (AND), `||` (OR), `!` (NOT), `??` (nullish coalescing)
- **Ternary**: `condition ? exprIfTrue : exprIfFalse`

### Control Flow

- `if / else if / else` — the basic branching construct
- `switch` — useful for multiple exact matches, uses strict comparison
- `for` — classic C-style loop
- `while` / `do-while` — condition-first vs condition-last
- `for...of` — iterate iterables (arrays, strings, maps, sets)
- `for...in` — iterate object enumerable keys (use with caution)

## Key Takeaways

- Always prefer `===` over `==` to avoid type coercion surprises
- `??` only checks for `null`/`undefined`, unlike `||` which checks for any falsy value
- `for...of` is cleaner than index-based `for` for arrays
